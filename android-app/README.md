# MeshTalk Android App

**Team ZEDOX | Imagine Cup 2025**

## Quick Start

1. Open Android Studio
2. File → Open → Select `android-app` folder
3. Wait for Gradle sync
4. Connect device (USB debugging ON)
5. Run → Run 'app'

## Requirements

- Android Studio 2023.x+
- Android device API 21+
- WiFi capability
- 4 phones for full testing
